## 如何撰写文章

本站使用 Markdown 格式撰写文章，支持：

- **数学公式**：使用 LaTeX 语法，如 `$E=mc^2$`
- **代码高亮**：支持多种编程语言
- **图表**：可嵌入图片或使用 Mermaid 绘制

::: tip 添加新文章
1. 在 `docs/articles/` 目录下创建新的 `.md` 文件
2. 在此页面添加文章链接
3. 推送到 GitHub 自动部署
:::
