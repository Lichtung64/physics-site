# 大学物理

## 课程简介

大学物理是理工科本科生的必修基础课程。本课程旨在建立扎实的物理概念和物理图像，培养学生的物理思维能力和解决问题的能力。

---

## 教学内容

### 第一章：质点运动学

- 位置矢量与位移
- 速度与加速度
- 切向加速度与法向加速度
- 相对运动

**核心公式：**

$$\vec{v} = \frac{d\vec{r}}{dt}, \quad \vec{a} = \frac{d\vec{v}}{dt}$$

### 第二章：质点动力学

- 牛顿运动定律
- 常见的力：重力、弹力、摩擦力
- 惯性系与非惯性系
- 惯性力

**牛顿第二定律：**

$$\vec{F} = m\vec{a}$$

### 第三章：动量与角动量

- 动量定理与动量守恒
- 质心运动定理
- 角动量定理与角动量守恒
- 碰撞问题

**动量守恒定律：**

$$\vec{p} = \sum_i m_i \vec{v}_i = \text{常数}$$

### 第四章：功与能

- 功与功率
- 动能定理
- 势能
- 机械能守恒定律
- 功能原理

**机械能守恒：**

$$E = \frac{1}{2}mv^2 + U = \text{常数}$$

### 第五章：刚体转动

- 刚体运动的描述
- 转动惯量
- 转动定律
- 角动量守恒
- 进动

**转动惯量：**

$$I = \int r^2 dm$$
