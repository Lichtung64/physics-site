# 课程总览

这里收录了我讲授的各门课程资料，包括讲义、习题、补充材料等。

---

## 课程列表

### [大学物理](/courses/general-physics)

面向本科生的大学物理课程，涵盖力学、热学、电磁学、光学和近代物理。

**主要内容：**
- 质点运动学与动力学
- 刚体转动
- 热力学基础
- 电磁场理论
- 波动光学
- 量子物理初步

---

### [群论](/courses/group-theory)

群论基础及其在物理学中的应用。

**主要内容：**
- 群的基本概念
- 置换群与点群
- 群表示论
- 李群与李代数
- 群论在量子力学中的应用

---

### [物理与AI](/courses/physics-ai)

探索人工智能技术在物理研究中的应用。

**主要内容：**
- 机器学习基础
- 神经网络与深度学习
- 数据驱动物理建模
- 物理信息神经网络 (PINN)
- 强化学习在物理问题中的应用

---

## 如何使用本站

1. 点击上方课程名称进入对应课程页面
2. 每门课程页面包含章节目录和资料下载链接
3. 讲义以 Markdown 格式呈现，支持数学公式渲染
4. PDF、PPT 等文件可在[资料下载](/resources)页面获取
