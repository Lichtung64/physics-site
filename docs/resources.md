## 添加新文件

1. 将文件放入 `public/files/` 目录
2. 在此页面添加对应的表格行
3. 链接格式为 `/files/文件名`

::: warning 文件大小限制
如果文件较大（>50MB），建议使用云盘链接。GitHub 有单文件 100MB 限制。
:::
